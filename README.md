# morm

morm is a node library for handeling reactive state

## Installation

Use the node package manager [npm](https://www.npmjs.com/) to install.

```bash
npm i @okkr/morm
```

## Usage

```typescript
import * as $ from "@okkr/morm";

console.log($);

```
